\subsection{Detailed Description}
The main matrix type. 

This is the main matrix structure used in the library. Most of the the time, you should expect to be given a pointer to a heap-\/allocated version of this structure.

The alias {\ttfamily matrix\+\_\+t} for this structure is defined for convienience.

To create a matrix, see the {\ttfamily matrix\+\_\+create} function.
